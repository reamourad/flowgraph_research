## Instructions to run RQ2

1. Run `sampling.ipynb` to clone the repos locally.

2. Run the notebook corresponding to the variant you want to evaluate:

   - **Baseline** — `claude_baseline.ipynb`
   - **AST only** — `claude_ast.ipynb`
   - **AST + prose thinking** — `claude_ast_with_thinking.ipynb`
   - **Full Flow graph** — `claude_ast_with_flowgraph.ipynb`
   - **Summary flow graph** — `claude_summary_flowgraph.ipynb`

3. Don't forget to change the `BATCH` variable in cell 2 to the one of your liking.
